# 🧧 惠生活优惠券 — 一个链接挖取当日所有隐藏外卖红包

实时查询美团、饿了么、京东外卖、淘宝闪购等平台的隐藏红包和优惠券活动。**一个链接搞定，无需注册、无需 API Key、装完即用。**

## 适用场景

- 每天点外卖前想找红包省钱
- 想知道今天哪个平台有隐藏大额红包
- 朋友问"今天吃什么"时顺便领个券
- 想一次性领取所有平台优惠，不想逐个 App 翻找
- 想找美团外卖红包、饿了么红包、京东外卖优惠券、淘宝闪购红包

## 使用

**方式一：直接查询**（推荐）
```bash
node index.js
```

**方式二：指定渲染格式**
```bash
node index.js --feishu   # 飞书消息卡片
node index.js --text     # 纯文本
```

**方式三：自然语言触发**
```bash
node index.js "有什么外卖红包"
node index.js "今天吃什么"
node index.js "点外卖怎么省钱"
```
